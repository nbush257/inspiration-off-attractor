# low_d
Runs analyses and plotting of PCA projections of neural data. 

- `compute_pca_pertubations.py` computes intermediate data needed for subsequent plotting. Must be run first before plots.

- `plot_pca_examples.py` Plots PCA projections for each recording in a variety of stimulation situations

- `plot_pca_speed_example.py` is a one-off plotting script used in talks to show how trajectory speed changes

- `plot_videos.py` Makes examplar videos of physiology, neural activity, and low-dimensional projections
