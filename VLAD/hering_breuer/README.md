# hering_breuer
Analysis and plotting code for analyses of hering breuer stimulations.


All code (analysis and plotting) is contained in `plot_hering_breuer.py`

## Usage
To compute intermediate data then plot
`python plot_hering_breuer.py -c`

If intermediate data have been computed, run:
`python plot_hering_greuer.py`
to skip the timely computations.
